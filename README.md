## Описание

Программная платформа, которая умеет (точнее "учится", на данный момент) работать с разными технологиями аватаров, от пнг-картинок до трёхмерных. Поддержка трекеров осуществляется таким же способом, с помощью плагинов.

## Возможности
Может:
* Запуститься
* Подключить плагины трёх видов
* Настроить интерфейс
* Поддерживать локализацию (точнее пытается)
* Открывать аватаров
* Показывать нагрузку на систему от программы
* Лагать
* Вызывать чувство нейрослопа
* Закрываться
* Писать краш-репорты
